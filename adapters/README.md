# adapters/

Platform adapters for distributing factor-echelon season packs to different hosts.

Populated by **Plan 01 (Foundation)** and extended by **Plan 04 (OpenClaw)** and **Plan 11 Phase E (Echelon)**.

Target adapters:
- `claude-code/` — Claude Code plugin format
- `openclaw/` — OpenClaw agent format
- `echelon/` — Echelon.app season pack format

See `docs/specs/2026-04-08-factor-echelon-design.md` for the full specification.
