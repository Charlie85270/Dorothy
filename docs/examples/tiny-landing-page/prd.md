# Landing Page for Acme Analytics

## Overview

Build a responsive marketing landing page for Acme Analytics, a new data visualization SaaS product. The page should convert visitors into trial signups.

## Goals

- Communicate the product value proposition immediately above the fold
- Showcase three pricing tiers (Free, Pro, Enterprise) with feature comparison
- Capture email signups with a contact form connected to a mailing list API
- Achieve a Lighthouse performance score of 95+ on mobile
- Deploy to Vercel with automatic preview deployments on PR

## User Stories

- As a visitor, I can understand what Acme Analytics does within 5 seconds of landing
- As a visitor, I can compare pricing tiers side-by-side and select one
- As a visitor, I can submit my email to start a free trial
- As a visitor, I can view the page comfortably on any device (mobile, tablet, desktop)

## Page sections

1. **Hero** -- headline, subheadline, CTA button, product screenshot
2. **Features grid** -- 6 feature cards with icons and descriptions
3. **Pricing table** -- 3 tiers with feature comparison and CTA per tier
4. **Testimonials** -- 3 customer quotes with photos
5. **Contact form** -- email, name, company, message fields
6. **Footer** -- links, social icons, legal

## Technology

- React with Next.js (App Router)
- Tailwind CSS for styling
- Vercel deployment
- Resend for contact form email delivery

## Timeline

2 weeks

## Notes

This is a straightforward frontend project. No backend beyond the contact form API route. The scope estimator should classify this as Medium tier (~11 characters).
